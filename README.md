# BancaMex - Sistema Bancario en Java Swing

Proyecto integral desarrollado para la carrera de **Ingeniería de Software**, enfocado en la simulación de operaciones bancarias mediante una interfaz gráfica de usuario (GUI) robusta y profesional.

## Funcionalidades Principales
- **Sistema de Autenticación:** Login validado mediante listas dinámicas de usuarios.
- **Gestión Financiera:** Consultas de saldo, depósitos y retiros con validación de fondos en tiempo real.
- **Transferencias Interbancarias:** Lógica de movimiento de capital entre diferentes objetos de usuario.
- **Arquitectura de Navegación:** Implementación de `CardLayout` para una experiencia de usuario fluida sin ventanas emergentes innecesarias.

## Stack Tecnológico
- **Lenguaje:** Java 17+ ☕
- **Librerías:** Swing & AWT para el diseño de la interfaz
- **Gestor de Dependencias:** Maven
- **Paradigma:** Programación Orientada a Objetos (POO) con fuerte uso de Encapsulamiento.
